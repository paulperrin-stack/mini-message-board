# Mini Message Board 💬

A simple message board application built with Express and EJS.

## 🎮 Live Demo

Coming soon.

## How to Use

- Visit the homepage to see all messages
- Click on a message to view its details
- Click "New Message" to add your own

## Features

- View all messages on the homepage
- Add new messages through a form
- View individual message details
- Dynamic routing using URL parameters

## What I Learned

- Setting up an Express server
- Using EJS for templating
- Handling GET and POST routes
- Working with route parameters (`req.params`)
- Processing form data with `req.body`
- Structuring a basic MVC-style app

## Built With

- Node.js
- Express
- EJS
